# Bank Telemarketing Success Prediction

## Project Overview

This project aims to predict whether a customer will subscribe to a bank term deposit based on information collected during telemarketing campaigns.
Using customer demographics, financial information, and previous campaign interactions, machine learning models were developed and evaluated to identify the factors that influence subscription decisions.

---

## Problem Statement

Banks spend significant resources conducting telemarketing campaigns. Contacting every customer is inefficient and costly.
The objective of this project is to build a predictive model that helps identify customers who are more likely to subscribe to a term deposit, enabling more targeted marketing efforts.

---

## Dataset

The dataset contains customer information such as:

* Age
* Job
* Marital Status
* Education
* Account Balance
* Housing Loan Status
* Personal Loan Status
* Contact Type
* Campaign Details
* Previous Marketing Outcomes

**Target Variable:**

* Subscription to Term Deposit (Yes / No)

---

## Project Workflow

### Data Preprocessing

* Handled missing values
* Cleaned and transformed raw data
* Encoded categorical variables
* Scaled numerical features where required

### Exploratory Data Analysis (EDA)

* Analyzed customer demographics
* Identified feature distributions
* Examined correlations and patterns
* Visualized trends using charts and plots

### Machine Learning Models

The following models were trained and evaluated:

* Logistic Regression
* Random Forest Classifier

### Model Evaluation

Models were compared using classification metrics to determine the best-performing approach for customer subscription prediction.
